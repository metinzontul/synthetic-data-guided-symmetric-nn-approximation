This repository contains the Python codes, synthetically generated numerical data, tables, and figures used in the manuscript “Synthetic Data-Guided Symmetric Neural Network Approximation in Banach Spaces”.

The repository supports the numerical experiments of the paper, including kernel diagnostics, fractional approximation examples, fractional Taylor remainder validation, and a controlled data-driven blind prediction experiment on a synthetic monthly temperature-like time series with seasonality, trend, and random noise. No external or real-world dataset is used.
